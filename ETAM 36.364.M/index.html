Рожковая кофеварка De'Longhi PrimaDonna XS ETAM36.364.M

Неизменно идеальный кофе каждый день

С кофемашиной De'Longhi PrimaDonna XS ETAM36.364.M вы сможете наслаждаться великолепным вкусом любимого напитка, не выходя из дома. Она позволяет приготовить насыщенный черный кофе и взбить молочную пенку для капучино, как в любимой кофейне.

НОВАЯ СИСТЕМА «LATTECREMA SYSTEM»

Кофемашина De'Longhi ETAM 36.364 M приготовит ваш любимый напиток с пышной молочной пенкой, которая обладает идеальной плотной и шелковистой текстурой. На панель выведены кнопки с названиями напитков: Caffelatte, Cappuccino, Latte Macchiato, Milk. Нажмите кнопку, и кофемашина быстро приготовит ваш заказ. Компактный контейнер для молока можно хранить в холодильнике.

НАСАДКА ДЛЯ ПОДАЧИ ПАРА И ГОРЯЧЕЙ ВОДЫ

Помимо устройства для автоматического приготовления капучино кофемашина комплектуется дополнительной насадкой для подачи пара. Вы сможете ней воспользоваться и для приготовления капучино, и для получения просто горячей воды для чая.

ФУНКЦИЯ ПРОГРАММИРОВАНИЯ

Благодаря функции программирования вы сможете один раз установить параметры приготовления любимого рецепта по своему желанию, а кофемашина ETAM 36.364M PrimaDonna XS DeLuxe воспроизведет его в следующий раз по памяти. Установите с помощью меню температуру напитка, объем воды на порцию, количество кофе и молока.

КНОПКА ПОДОГРЕВА ЧАШЕК

Кофемашина имеет специальную кнопку для включения подогрева чашек. Вы сможете воспользоваться этой функцией, когда это будет необходимо. Включите кнопку, поставьте посуду на специальную панель. Кофе приготовленный в прогретую чашку сможет еще глубже раскрыть свой вкус и аромат, напиток долгое время будет сохранять свою температуру.

ПРОСТОЕ ОБСЛУЖИВАНИЕ

Периодически необходимо проводить чистку различных элементов в кофемашине. В данной модели все сделано для максимально удобного обслуживания. Блок заваривания легко вытягивается из устройства, для его очистки достаточно промыть его под струей воды. Кофемашина подскажет, когда придет время произвести очистку от накипи. Устройство для вспенивания молока легко очищается нажатием кнопки «Clean».
